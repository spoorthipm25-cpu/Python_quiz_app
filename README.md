# 🧠 Python Quiz App

An interactive and visually appealing **Quiz Application** built using **Streamlit**.
This app tests Python knowledge across different difficulty levels with a timer and downloadable results.

## 📌 Features

* 🎯 **Multiple Difficulty Levels**

  * Basic 
  * Intermediate
  * Advanced

* ⏱️ **Timer Per Question**

  * Countdown timer for each question
  * Auto-moves to next question when time is up

* 🧠 **Step-by-Step Quiz Flow**

  * One question at a time
  * Clean card-style UI

* 📊 **Instant Result & Feedback**

  * Score displayed after completion

* 📄 **Download Results**

  * Export quiz results as CSV

* 🎨 **Modern UI**

  * Soft gradient background (sky blue + light pink)
  * Clean and readable design

## ▶️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/spoorthipm25-cpu/python-quiz-app.git
cd python-quiz-app
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run app.py
```
